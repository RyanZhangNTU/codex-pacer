# Codex Pacer v1.0.0

## Summary

`v1.0.0` is the first stable public release of **Codex Pacer**.

This release establishes the public product line around a local-first desktop app for understanding Codex usage as pace, value, and session-level activity.

## Highlights

- first stable public release of Codex Pacer
- local-first import of Codex usage data from `~/.codex` or a custom `CODEX_HOME`
- local SQLite indexing for overview analytics and drill-down workflows
- API-equivalent value estimation and subscription payoff tracking
- rolling quota tracking, including `5-hour` and `7-day` pacing when available
- conversation-level breakdowns across root sessions, subagents, models, and token metrics
- macOS menu bar integration for quick quota snapshots
- refreshed bilingual documentation for installation, packaging, and release guidance

## Packaging

Official public release asset:

- signed and notarized macOS Apple Silicon DMG via GitHub Releases

## Notes

- `v1.0.0` was the first stable release line. See the latest release notes for the current stable version.
- Intel macOS, universal builds, Windows, Linux, and auto-update delivery are not currently promised as official release assets.
- Codex Pacer remains local-first and does not depend on a cloud sync service to work.
